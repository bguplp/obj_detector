# obj_detector
Object detection base yolo4-tiny for armadillo2. 
Face recognizer. 


## Dependencies
* ubuntu 16.04 LTS
* ROS kinect
* python 2.7.*
* [bguplp/gazebo_worlds package](https://github.com/bguplp/gazebo_worlds)

The following python2 packges are required:
* OpenCV
* scipy
* numpy
* cmake
* face_recognition
